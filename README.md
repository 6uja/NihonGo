# NihonGo

JavaFX와 SQLite를 활용하여 개발한 일본어 학습 퀴즈 애플리케이션

## 프로젝트 소개

NihonGo는 일본어 학습자가 퀴즈를 통해 단어와 문법을 학습하고 자신의 학습 수준을 확인할 수 있도록 개발한 프로그램이다.

단순히 문제를 풀고 정답을 확인하는 기능에서 끝나는 것이 아니라 사용자 관리, 오답노트, 랭킹, 학습 통계 기능을 포함하여 학습 데이터를 지속적으로 관리할 수 있는 종합 학습 시스템으로 설계하였다.

---

## 개발 목적

* 일본어 단어 및 문법 학습 지원
* 학습 기록 저장 및 관리
* 오답 복습 기능 제공
* 학습 성과 분석 기능 제공
* 객체지향 설계 및 데이터베이스 활용

---

## 주요 기능

### 회원 관리

* 회원가입
* 로그인
* 사용자 정보 저장

### 일본어 퀴즈

* 초급 / 중급 / 고급 카테고리
* 객관식 문제
* 빈칸 채우기 문제
* 10문제 랜덤 출제
* 문제당 20초 제한

### 오답노트

* 틀린 문제 자동 저장
* 정답 및 사용자 답안 확인
* 복습 기능 제공

### 랭킹 시스템

* 최고 점수 저장
* 카테고리별 랭킹 제공

### 학습 통계

* 카테고리별 정답률 분석
* 학습 결과 시각화

### 문제 관리

* CSV 기반 문제 추가
* 관리자 모드를 통한 문제 등록
* SQLite 데이터베이스 저장

## 주요 기술

### JavaFX

직관적인 GUI 환경을 제공하기 위해 JavaFX를 사용하였다.

초기에는 Java Swing을 고려하였으나 보다 현대적인 사용자 인터페이스를 구현하기 위해 JavaFX를 선택하였다.

### SQLite

사용자 정보, 문제 데이터, 오답노트, 랭킹 데이터를 저장하기 위해 SQLite를 사용하였다.

### 객체지향 설계

기능별 클래스를 분리하여 유지보수성과 확장성을 향상하였다.

* UserManager
* QuestionManager
* RankingManager
* StatsManager
* DatabaseManager

### CSV 연동

문제 데이터를 CSV 파일로 관리하여 프로그램 수정 없이 새로운 문제를 추가할 수 있도록 구현하였다.
